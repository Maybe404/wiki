快速上手指南

按照以下步骤完成初始配置，大约需要 15 分钟。完成后你将拥有一个可用的本地开发环境。

前置条件：请确保已安装 Node.js 18+、Git 2.40+ 和 Docker Desktop 4.x。 可在终端运行 node -v 确认版本。

操作步骤

步骤 1：克隆仓库

使用以下命令将项目代码克隆到本地。推荐使用 SSH 协议，避免频繁输入密码。 

git clone git@github.com:your-org/your-project.git
cd your-project

克隆成功后，进入项目目录。后续所有命令均在此目录下执行。

步骤 2：安装依赖

运行以下命令安装项目依赖。首次安装可能需要 2–3 分钟，具体取决于网络速度。 

npm install

安装完成后，node_modules 目录约占用 200MB 磁盘空间。

步骤 3：配置环境变量

复制示例配置文件并填写本地环境的密钥。不要将 .env 提交到版本控制。 

cp .env.example .env
# 编辑 .env，填写以下字段：
# DATABASE_URL=postgresql://localhost:5432/mydb
# SECRET_KEY=your-secret-key-here
# API_ENDPOINT=https://api.example.com

步骤 4：启动开发服务器

运行以下命令启动本地开发服务器。成功后在浏览器访问 http://localhost:3000。 

npm run dev

# 预期输出：
# ✓ Ready in 2.1s
# ○ Local: http://localhost:3000

开发服务器支持热重载，修改代码后无需手动重启。

验证安装

打开浏览器访问以下地址，验证各功能是否正常：

地址 预期结果
http://localhost:3000 显示主页
http://localhost:3000/health 返回 {"status":"ok"}
http://localhost:3000/admin 显示登录页

遇到问题？查看 logs/dev.log 获取详细错误信息，或在 Slack 的 #dev-help 频道提问。
