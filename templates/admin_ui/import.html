HTML 模板规范

推荐使用的 class

  • article-shell — 文章外层容器
  • info-callout — 提示/注意框
  • content-card — 内容卡片
  • code-block — 代码块
  • table — 表格

可编辑标记规则

  • 未标记时，系统会自动识别标题、段落、列表与表格文本
  • 需要精确控制时，加 data-editable="true"
  • 同时加 data-editable-id="唯一ID"
  • ID 全文唯一，只用字母/数字/连字符

严禁事项

  • 禁止内联 style 属性
  • 自动移除 <script>、<iframe>、<style>
  • 自动移除 <link> 等外部资源声明
  • 禁止 on* 事件属性
