HTML 导入策略

原样保留

  • 保留 <style>、<script>、字体与图形资源声明
  • 不把外部页面改写成 Atlas 正文样式
  • 不注入 data-editable，导入后作为完整页面展示

基础检查

  • 上传文件需为 .html 或 .htm
  • 单次导入建议 1 KB 至 2 MB，超出范围会提示错误
  • 默认要求能抽取到可见文本；纯可视化页面需主动勾选说明

隔离策略

  • 阅读页通过 iframe sandbox 展示完整 HTML
  • 文档页面使用独立 CSP，默认只允许常见 CDN、字体、图片与 HTTPS 请求
  • 父页面只接受来自对应 iframe 的高度消息
